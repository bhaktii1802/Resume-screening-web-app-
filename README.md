
# 📄 Resume Screening App

A machine learning-powered web app built with Streamlit that predicts the job category of a resume (PDF, DOCX, or TXT). It simplifies the recruitment process by automatically categorizing resumes using NLP and a pre-trained SVM model.



## Features

- 📁 Upload resumes in PDF, DOCX, or TXT formats
- 🧹 Auto text extraction and cleaning
- 📊 Predicts job category using an SVM classifier
- 🌐 Easy-to-use Streamlit interface
- 🧠 Powered by TF-IDF vectorization and scikit-learn

## Tech Stack

- **Frontend/UI**: Streamlit  
- **Backend**: Python  
- **Machine Learning**: scikit-learn (SVM Classifier)  
- **NLP**: TF-IDF Vectorizer  
- **File Handling**: PyPDF2, python-docx  
- **Model Storage**: Pickle
